# 📚 OkuPaylaş

OkuPaylaş, kitap severlerin kitaplarını güvenli şekilde alıp satabilecekleri ve takas edebilecekleri bir platformdur.

## Özellikler
- 📖 Kitap alım-satımı
- 🔄 Kitap takası
- 💳 Emanet ödeme sistemi
- ⭐ Kullanıcı puanlama sistemi
- 💬 Mesajlaşma
